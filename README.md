# super-baby-cli


> 我的cli


## Quick Overview

```sh
npx @wuge/create-project-cli new my-app
cd my-app
yarn dev
```

_npx 依赖 npm v5.2 及更高版本_

## 新建项目

**依赖 node 10.0.0+**

目前只支持生成vue项目，其他的之后再做

**npx（推荐）**

```sh
npx @wuge/create-project-cli new my-app
```

_需要 npm 5.2+ 版本_

**npm**

```sh
npm install @wuge/create-project-cli -g
wuge-create my-app
```

_需要 npm 6+ 版本_

**yarn**

```sh
yarn add @wuge/create-project-cli -g
wuge-create my-app
```

_需要 yarn 0.25+ 版本_
